Student Marks Analyzer (NumPy)

A Python project built using NumPy to analyze student marks and generate useful academic statistics.

Features
Input marks for multiple students
Calculate total marks
Calculate percentage
Assign grades (A, B, C, D, F)
Pass/Fail status
Highest scorer
Lowest scorer
Class average
Subject-wise toppers
Student ranking

Technologies Used
Python 3
NumPy

How to Run
Clone the repository.
Open the project folder in VS Code.
Install NumPy (if not installed):
pip install numpy
Run the program:
python marks_analyzer.py
